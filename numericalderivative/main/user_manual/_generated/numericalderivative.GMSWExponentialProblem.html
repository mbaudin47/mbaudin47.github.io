numericalderivative.GMSWExponentialProblem¶

class numericalderivative.GMSWExponentialProblem¶

Create an exponential derivative benchmark problem

See eq. 4 page 312 in (Gill, Murray, Saunders & Wright, 1983)

The function is:

\[f(x) = \left(\exp(x) - 1\right)^2 + \left(\frac{1}{\sqrt{1 + x^2}} - 1\right)^2\]

for any \(x\). The test point is \(x = 1\). The optimal finite difference step for the forward finite difference formula of the first derivative is approximately \(10^{-3}\).
